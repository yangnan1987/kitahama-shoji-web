## 5. 建站时备忘（非法律建议）

- 日本不动产中介站点常见页：`会社概要`、`事業内容`、`取扱物件（一覧・詳細）`、`お問い合わせ`、**宅建業者表示**（法令で求められる項目をページ内に明記）、`プライバシーポリシー` 等。具体条文与必須表記请咨询宅建士或律师。  
- 静态多页：可考虑 Eleventy、Astro、或纯 HTML 多页 + 共通ヘッダー／フッター。  
- 字体与配色：日文明朝体／ゴシック系 Web フォント、低彩度アクセント、footer に上記表示ブロックを固定する構成が一般的。  
- **Canonical**：对外宜统一使用 **`https://kitahamashoji.com/`**（是否含 `www` 择一并在注册商／托管处做重定向）。
